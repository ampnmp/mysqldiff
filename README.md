# mysqldiff
比较MySQL数据库表结构

<p>    
比较两个MySQL数据库全部表结构的差异，<br>
包括表名、存储引擎、字符集、注释的不同，<br>
以及每张表中的字段名、数据类型、字符集、默认值、注释的不同，<br>
还有索引的不同、字段顺序的不同。<br>
</p>
